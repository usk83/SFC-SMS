シフト管理システム for ヰベント企画局
==========================================

2015年度 秋学期
スクリプト言語プログラミング 最終課題

## 概要

ヰベント企画局のシフト作成効率化のためのアプリケーション  
シフト作成のための情報入力、シフト作成、シフト管理をすべてアプリケーション上で行えるようにする

## 機能

おおまかに以下の機能を持つ。

- プロフィールの設定
- タスクの追加、管理
- シフト作成のためのアンケート回答
- シフト確認

またシフト管理者はさらに以下の機能を持つ。

- シフト作成
- シフトのための項目追加
